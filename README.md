strategy >> performance and change explorer

clean install
-------------
* install git
* install nodeJS (e.g. ubuntu = sudo apt-get install -y nodejs)
* install mongodb
* npm install -g bower
* npm install -g gulp


checkout space from git
--------------------
* git clone https://github.com/gkathan/space
* cd space
* npm install 
* bower install
* npm run-script startDEV
